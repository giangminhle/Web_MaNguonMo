# 📁 Document Folder

## 📋 Nội Dung

Folder này chứa các tài liệu và file liên quan đến dự án QuanLySan333.

## 🗂️ Cấu Trúc

### 📦 SOUCRE+BACKUP/
- `quanlysan_db.backup` - Database backup file (66KB)

### 🎥 VIDEO CÀI ĐẶT + File/
- `HDSD_Web.md` - Hướng dẫn sử dụng web
- `README.md` - File readme cũ

### 🎬 VIDEO DEMO/
- `1.mp4` - Video demo phần 1 (22MB)
- `2.mp4` - Video demo phần 2 (19MB)  
- `3.mp4` - Video demo phần 3 (9MB)

> **Lưu ý:** Các file video đã được loại bỏ khỏi Git do giới hạn 100MB của GitHub.

### 📄 Word báo cáo/
- `Word báo cáo.docx` - File báo cáo (10MB)

> **Lưu ý:** File Word đã được loại bỏ khỏi Git do giới hạn 100MB của GitHub.

## 🚀 Lưu Ý Quan Trọng

### Files Đã Được Loại Bỏ (.gitignore)
- **Video files:** `*.mp4`, `*.avi`, `*.mov`, etc.
- **Large documents:** `*.docx`, `*.pdf`, `*.pptx`, `*.xlsx`
- **Database backups:** `*.backup`, `*.sql`, `*.dump`

### Files Vẫn Được Lưu Trữ
- `quanlysan_db.backup` - Database backup (dưới 100MB)
- Các file markdown documentation

## 📥 Cách Lấy Files Lớn

Nếu bạn cần các file lớn đã bị loại bỏ:

1. **Video files:** Yêu cầu trực tiếp hoặc sử dụng link chia sẻ
2. **Word báo cáo:** Yêu cầu trực tiếp hoặc sử dụng link chia sẻ
3. **Database backup:** Có sẵn trong folder SOUCRE+BACKUP/
